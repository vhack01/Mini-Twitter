# Mini Twitter Clone

## Create app

    - npx create-create-app twitter-clone

## Dependencies

    - Tailwindcss
    - React-Router-Dom
    - React-icons
    - Axios
    - React Hot Toast
    - Redux
    - Redux-persist

### backend dependencies

    - Express
    - Mongoose
    - Dotenv
    - Json Web Token(JWT)
    - Cors
